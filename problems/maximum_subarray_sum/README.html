H. Maximum Subarray Sum
time limit per test
2.0 s
memory limit per test
256 megabytes
input
standard input
output
standard output

Given an array of nn integers, your task is to find the maximum sum of values in a contiguous, nonempty subarray.

Input

The first input line has an integer n: the size of the array.

The second line has nn integers x1,x2,…,xnx1,x2,…,xn: the array values.

Constraints:

  • 1≤n≤2⋅1051≤n≤2⋅105
  • −109≤xi≤109−109≤xi≤109
Output

Print one integer: the maximum subarray sum.

Example
Input
Copy
8
-1 3 -2 5 3 -5 2 2
Output
Copy
9
